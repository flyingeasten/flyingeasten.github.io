## 为什么选择 Hugo？
- 速度快，生成静态页面效率高
- 主题丰富，LoveIt 主题美观且功能全
- 支持 Markdown，写作体验好

代码示例：
```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, Hugo!")
}
````
